## 古い概要

- WebGLの練習用リポジトリです。
	- TypeScript+Webpack導入と、npm run deployまでのセットアップなど環境整備のサンプルとしても。

- WebGL操作ライブラリにはthree.jsを選択しました。

- npm installのあとは、npm run devでローカルサーバ起動します。

- デプロイは`npm run deploy`で実行できますが、S3へのデプロイに必要な機密情報はリポジトリに含んでいません。
